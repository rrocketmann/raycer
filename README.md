Open-source AI gym for reinforcement learning research.

Built with:
- **Bevy** – game engine (rendering, ECS, audio, input)
- **avian3d** – physics (rigid bodies, colliders, gravity, friction)
- **bevy_egui** – debug overlay UI (speed, controls, telemetry)
- **bevy_light** – directional lighting & shadow mapping
